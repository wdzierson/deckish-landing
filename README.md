# Deckish Landing Page

Builder-first cyberdeck community landing page. Built with Vite + TypeScript + React.

## Quick Start

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Import this repo at [vercel.com/new](https://vercel.com/new)
2. Vercel auto-detects Vite — no config needed
3. Hit Deploy

## Stack

- Vite
- TypeScript
- React 18
- CSS custom properties (no framework)

## Sections

- **Hero** — tagline + dual CTAs
- **Value Prop** — what Deckish is and why it exists
- **Features** — build logs, parts browser, gallery, Discord gateway, build plans, profiles
- **Discord Gateway** — top cyberdeck communities
- **CTA + Footer**
